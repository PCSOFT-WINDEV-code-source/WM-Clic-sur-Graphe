  
<span style="font-family:Arial sans-serif;font-size:16px;">WM Clic sur Graphe</span>

  
<span style="font-family:Arial sans-serif;font-size:14px;">L'exemple "WM Clic sur Graphe" propose une solution pour excentrer les parts d'un graphe de type secteur sur un simple clic utilisateur.</span>

<span style="font-family:Arial sans-serif;font-size:14px;">Cet exemple met à disposition un modèle de champs "MDLC\_ExcentreGraphe". Ce modèle de champs permet d'activer l'excentration de parts sur simple clic sur un graphe.</span>

<span style="font-family:Arial sans-serif;font-size:14px;">Pour disposer de cette fonctionnalité dans une autre application, il suffit d'importer le modèle de champs "MDLC\_ExcentreGraphe". </span>
